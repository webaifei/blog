# 后台架构

```
+ node_modules/ 依赖文件
+ views         模版文件
+ public        静态资源文件
  - css
  - scripts
  - images
+ utils         辅助文件(功能函数集合)
+ db            数据库连接
+ model         集合封装
+ controllers   具体的业务逻辑处理
+ routes        路由映射
+ test          单元测试代码
+ app.js        程序入口文件
+ config.js     公共配置文件
```
